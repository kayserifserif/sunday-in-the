# nanogenmo-2022

## Install

Install [Miniconda](https://docs.conda.io/en/latest/miniconda.html).

Install [spaCy](https://spacy.io/usage) (instructions below for macOS):

```
conda install -c conda-forge spacy
python -m spacy download en_core_web_sm
```

## Setup

```
conda create -n venv
conda activate venv
```

## Running

```
python main.py
```
